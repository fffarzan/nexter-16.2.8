# Font Module

automatically optimizes your fonts and removes external network requests.

built-in self-hosting for any font file. CSS and font files are downloaded at build time and self-hosted with the rest of your static assets. --> load web fonts with no layout shift.

## Google Fonts

automatically self-host. no requests are sent to Google by the browser.

Note: If loading a variable font, you don't need to specify the font weight.
